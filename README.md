Auto-Form is a smart personal data vault and autofill web application designed to eliminate repetitive form filling. Store your personal, academic, and professional information once, and use it to automatically populate applications for jobs, schools, scholarships, and more.

Built as a lightweight HTML, CSS, and JavaScript MVP, Auto-Form focuses on simplicity, speed, and privacy while laying the foundation for a scalable automation platform.
